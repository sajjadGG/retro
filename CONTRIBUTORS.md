## Contributing

Contributions are welcome! If you'd like to help:

1. Fork the repository
2. Create a feature branch
3. Make your changes and add tests
4. Run the test suite: `pytest tests/ -v`
5. Run linting: `ruff check src/retro/ tests/`
6. Run type checking: `mypy src/retro/`
7. Open a pull request

Please see the [README](README.md) for architecture details and the [specs/](specs/) directory for design documents.
